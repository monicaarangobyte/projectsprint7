# projectsprint7
Project Sprint 7 (Tripleten)
Descripción del proyecto

Este proyecto consiste en una aplicación web interactiva desarrollada con Streamlit, cuyo objetivo es facilitar la exploración visual de un conjunto de datos de autos (car_data).

La aplicación permite cargar los datos desde un archivo CSV y visualizar información clave de forma sencilla.

Funcionalidades principales

La aplicación web ofrece las siguientes funcionalidades:

Visualización de una vista previa del conjunto de datos para entender su estructura.

Generación de un histograma interactivo mediante un botón, que permite analizar la distribución de una variable del dataset.

Generación de un gráfico de dispersión interactivo mediante un botón, que permite explorar la relación entre dos variables.

Uso de gráficos interactivos creados con Plotly Express, integrados directamente en la aplicación.

Tecnologías utilizadas

Python

Streamlit para la creación de la aplicación web

Pandas para la manipulación de datos

Plotly Express para la visualización interactiva
